= 로그인 메시지 구성
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
모든 사용자가 로그인할 때 팝업 메시지로 표시할 메시지를 설정할 수 있습니다. 다음 로그인 시에만 표시할지, 아니면 이후 모든 로그인 시에 표시할지 선택할 수 있습니다.

관리자는 Admin > Notifications의 "Login Banner" 탭에서 메시지를 설정할 수 있습니다.

이 메시지는 기본적인 Markdown 형식을 지원하므로 굵게, 기울임꼴, 표 등의 서식을 적용할 수 있습니다.

사용자가 다음 로그인 시에만 메시지를 볼지, 아니면 이후 모든 로그인 시에 메시지를 볼지 선택하세요.

image:login_banner_settings.png["로그인 배너 설정"]

사용자는 로그인 메시지를 닫으려면 반드시 확인해야 합니다. 메시지를 닫으면 추가 팝업(새로운 기능 업데이트, 서비스 약관 업데이트 등)이 표시됩니다.
